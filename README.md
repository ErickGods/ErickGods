Sou Engenheiro de Automação (em formação) com atuação em **Quality Assurance e Engenharia de Dados na Hyundai Motor Brasil**. No dia a dia, construo pipelines de dados, desenvolvo automações com Power Platform e analiso indicadores de qualidade de campo para uma das maiores montadoras do mundo.

Paralelamente, estou em transição ativa para a área de **Data Engineering** — estudando Python, SQL, PySpark e arquitetura de dados em nuvem, com projetos reais que testam o que aprendo.

Fora do trabalho, desenvolvo o **Vault Inc.**, uma plataforma multi-agente com IA que simula uma empresa de tecnologia completa, usando LLMs, MCPs e pipelines automatizados.

## O que estou construindo

### 🏗️ Vault Inc. — Multi-Agent AI Company
> Empresa de tecnologia simulada com agentes de IA especializados

Uma plataforma multi-agente onde cada "funcionário" é um LLM com contexto, memória e responsabilidades definidas. Arquitetura com 4 domínios (Produto, Engenharia, Segurança e Dados/IA), ~10 agentes e um fluxo padronizado de intake de projetos via PM.

Stack: **Claude · MCPs · Python · VSCode · Obsidian**

---

### 📊 Em exploração ativa
- Pipelines ETL/ELT com Python e PySpark
- Arquitetura de dados moderna: Data Lakehouse, Delta Lake, Medallion Architecture
- Orquestração com Apache Airflow e Azure Data Factory
- CI/CD aplicado a engenharia de dados (GitHub Actions)
